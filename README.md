### 🔍 ¿Cómo encontrar los nombres exactos de los iconos?

Para asegurarte de que el icono aparezca, debes usar el nombre exacto que está registrado en la librería **Simple Icons**.

1. Ve a [simpleicons.org](https://simpleicons.org/).
2. Busca la tecnología que necesitas (ej. "SQL Server").
3. El nombre que debes poner en `logo=` es el **slug** (normalmente el nombre en minúsculas, a veces separado por guiones).

**Ejemplos comunes:**
* **SQL Server:** `microsoft-sql-server`
* **Next.js:** `next.js`
* **Ruby on Rails:** `ruby-on-rails`
* **C#:** `c-sharp`
